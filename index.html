Sample Data Files

                

1. Raw Call Transcripts (data/raw_transcripts.json)

        
[
  {
    "caller": "Provider",
    "transcript": "Hi, I'm calling to verify benefits for patient John Doe, DOB 01/01/1990.",
    "insurer_response": "Coverage is active. Deductible $500 with $200 met."
  }
]
                

2. Labeled NLP Data (data/labeled_intents_entities.json)

        
[
  {
    "text": "Coverage is active. Deductible $500 with $200 met.",
    "intent": "verify_benefits",
    "entities": [
      {"entity": "@coverage_status", "value": "active", "start": 0, "end": 6},
      {"entity": "@deductible_total", "value": "$500", "start": 25, "end": 29}
    ]
  }
]
   
   
       

Getting Started

       

Clone the Repository

        
git clone https://github.com/JSoul13/insurance-verification-ai.git
cd insurance-verification-ai
                

Run the Sample Script

        
python scripts/load_data.py
       

Output should show: First example: Coverage is active...
